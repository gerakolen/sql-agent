# HomoSQL Agent

## Описание проекта

Интеллектуальный ассистент для взаимодействия с SQL-базами данных через естественный язык, который автоматически преобразует пользовательские запросы в SQL-запросы и возвращает результаты в понятном формате. Система позволяет пользователям без технических навыков получать доступ к данным и аналитике из реляционных баз данных.

## Бизнес-ценность

Проект решает проблему доступа к данным для нетехнических пользователей, позволяя бизнес-аналитикам, менеджерам и другим специалистам получать ответы на вопросы о данных без необходимости писать SQL-запросы. Это ускоряет процесс принятия решений, снижает нагрузку на IT-команду и делает данные более доступными для всей организации.

## Видение MVP (Minimum Viable Product)

MVP представляет собой веб-приложение с консольным интерфейсом, которое:
- Принимает вопросы на естественном языке (русский/английский)
- Генерирует корректные SQL-запросы на основе схемы базы данных
- Выполняет запросы к PostgreSQL/SQLite базе данных
- Возвращает результаты в структурированном формате (таблица, JSON)
- Обрабатывает ошибки и предоставляет понятные сообщения

## Ключевой функционал MVP

### 1. Подключение к базе данных
- Поддержка PostgreSQL и SQLite
- Автоматическое чтение схемы базы данных (таблицы, колонки, типы данных, связи)

### 2. Обработка естественного языка
- Понимание вопросов на русском и английском языках
- Генерация SQL-запросов (SELECT, JOIN, WHERE, GROUP BY, ORDER BY)
- Контекстное понимание предыдущих запросов

### 3. Выполнение запросов
- Безопасное выполнение SQL-запросов
- Лимитирование результатов для предотвращения перегрузки
- Обработка ошибок базы данных

### 4. Форматирование ответов
- Представление результатов в виде таблицы
- Опциональное форматирование в JSON
- Естественно-языковое описание результатов

### 5. Интерфейс взаимодействия
- Консольный интерфейс для быстрого тестирования
- Возможность расширения до веб-интерфейса

## Технологический стек

- **Язык программирования**: Python 3.10+
- **LLM-фреймворки**: LangChain, LangGraph
- **Базы данных**: PostgreSQL, SQLite
- **ORM**: SQLAlchemy
- **AI-модели**: OpenAI GPT-4 / локальные модели (Ollama)

## Быстрый старт

```bash
# 1. Клонирование репозитория
git clone https://github.com/gerakolen/sql-agent.git
cd sipi-project

# 2. Создание виртуального окружения
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# или
.venv\Scripts\activate     # Windows

# 3. Установка зависимостей
pip install -r requirements.txt

# 4. Настройка переменных окружения
cp .env.example .env
# Отредактируйте .env с вашими настройками

# 5. Запуск
python src/main.py
```
